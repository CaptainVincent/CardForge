# cards

建立好的信用卡規則 JSON。每個檔可在 CardForge「資料 ▾ → 匯入」直接載入;匯入支援**一次選多個檔案 / 多個網址**,並可選**取代當前**或**加入現有**(把多張卡疊到同一張畫布)。

## 檔名慣例

```
<bank>-<card>-<period>.json
```

- 全英文、小寫、連字號分隔。
- `<period>` 後綴 = 該卡回饋**適用的週期**:

| 後綴 | 意義 |
|---|---|
| `2026h1` / `2026h2` | 上 / 下半年度活動檔期 |
| `2026` | 全年度活動 |
| `annual` | 全年一次(年度循環) |
| `monthly` | 月度循環(常態) |
| `once` | 一次性 |
